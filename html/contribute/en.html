How to contribute

If you like this extension and want to contribute, there are different ways:

  • Translations: So far, the extension is available in five languages (English, French, Spanish, Polish, and Japanesse), and it would be great if more languages are added. Do you speak Italian, German, ...? Do not hesitate and contribute to the internationalization of the extension. I've created a GitHub repository with the translatable texts. The URL is https://github.com/rumoroso/headingsmap_translations/tree/main. If you have any doubt or any question, you can send an email to the address that I created for the extension: headingsmap@gmail.com.
  • Feed-back: Another way to contribute is by sharing your feedback, ideas for improvements, reporting bugs or unexpected behavior, etc. Many of the improvements and fixes that the extension had were proposed by users. You can send any comment that you would like to share about the extension and about what it does or how it works. You can do it by writing to the email address headingsmap@gmail.com.
  • Sharing it: If you like the extension, you can also share it (with your colleagues, on social networks,...)
