プライパシーポリシー

このプライバシーポリシーは HeadingsMap ブラウザ拡張機能に適用され、機能するために必要な権限とそれらが必要な理由についてユーザーに通知することを目的としています。

データを収集しません

HeadingsMap はいかなる Web サイトからもデータを収集しません。 バニラ JavaScript を使用して (サードパーティのフレームワーク ライブラリを使用せずに) ブラウザ側 (クライアント側) で HTML 分析のみを実行し、Web ドキュメントからデータを収集、保存、共有しません。

これらは、開発者、コンサルタント、品質監査人、またはその他のエンド ユーザーの日常的な使用/作業を支援およびサポートするために開発され、維持されています。 隠された意図はありません。

ブラウザの権限

HeadingsMap は、ブラウザーによって現在表示されているページの HTML (HTML タグから直接派生したコンテンツ、または iframe の使用によって含まれるドキュメント) にアクセスします。 このため、また、保存できるオプションと設定のセット (テーマ、パネルの位置、分析の種類、エラーの識別など) をユーザーに提供するため、機能するにはブラウザーからの特定の許可が必要です。

拡張機能に必要なアクセス許可は Web サイトからデータを収集するものではありません。 これらのツールは HTML の解析を行い、ユーザーに HTML とその要素に適用されたスタイルに関する情報を提供し、分析の方法や結果の表示方法を定義するためのオプションセットをユーザーに提供する必要があります。 コンテンツからのデータ収集は行われず、ユーザーが拡張機能を使用するサイトに関するデータ収集も行われません。 また、拡張機能は実行されていない間は何の動作も行いません。

以下は、ブラウザーの権限とそれらが必要な理由のリストです。

all_urls
この権限により、拡張機能は許可されたスキームを使用するすべての URL に一致するため、任意の URL で動作することを可能にします。
activeTab
activeTab 権限はユーザーがいるサイトへの一時的なアクセスを許可し、拡張機能が現在のタブで "tabs" 権限を使用できるようにします。 この権限はインストール時に警告が表示されません。 これは拡張機能によって次の目的で使用されます。
  • 現在のタブで tabs.executeScript と tabs.insertCSS を呼び出します。 DOM を分析するにはスクリプトの実行が必要で、パネルのスタイルを設定して要素を強調表示するには CSS の挿入が必要です (拡張機能が備え、パネル内の情報を選択するときにページ内の要素を強調表示する機能)
  • tabs.Tab オブジェクトを返す API を介して、タブの URL とタイトルを取得します。 この情報を取得する唯一の理由は、ユーザーが見ている HTML ドキュメントを表示し、どのドキュメントで分析を実行する必要があるかを (タイトルを表示するカスタムセレクターで選択することにより) 決定するオプションを提供することです。
storage
拡張機能が (リモートの場所やサーバーではなく) ブラウザーにデータを保存および取得し、保存されたアイテムの変更をリッスンできるようにします。 この機能は、パネルの配置、テーマ、エラー警告などのユーザー設定を保存および取得するために拡張機能によって使用されます。
webNavigation
ナビゲーションのさまざまな段階にイベントリスナーを追加します。 ナビゲーションは、通常 (常にではありませんが) リンクのクリックやロケーションバーへの URL の入力などのユーザーアクションに応答して、ある URL から別の URL に遷移するブラウザー内のフレームで構成されます。
