14/3/2023 (v 4.2.0)
Ajout des raccourcis clavier pour basculer les nœuds de l'arborescence par niveau de titre. Sur Windows, utilisez la combinaison de la touche "alt" avec un numéro de touche pour réduire (ou développer) les nœuds de l'arborescence correspondant aux titres du même niveau que le numéro. Par exemple, "alt + 2" réduit (ou développe) tous les nœuds créés à partir des éléments h2. Sur Mac, la combinaison est "ctrl + cmd" avec le niveau de titre. Ce comportement est uniquement disponible pour l'arborescence "headingsMap" mais pas pour l'arborescence "HTML5 outline".
Correction du problème des styles incorrects dans le panneau lorsqu'il est dans un nouvel onglet qui n'est pas directement mis au premier plan (par exemple, lorsqu'un lien est ouvert dans un nouvel onglet).
13/3/2023 (v 4.1.0)
Lors de l'ouverture du panneau, la marge de l'élément BODY est conservée. De cette façon, il y a un espace entre le panneau et le contenu.
7/3/2023 (v 4.0.0)
Il y a une nouvelle option pour ouvrir automatiquement l'extension. Elle peut être configurée pour s'ouvrir toujours ou uniquement dans le domaine de la page active. La façon de l'utiliser est via un menu (avec l'icône "épingler") inclus dans la barre supérieure et qui propose les deux options accessibles avec le bouton-icône "épingler".
Maintenant, l'option "Changer la position du panneau" a son propre bouton dans la barre supérieure.
Le menu des paramètres est également accessible directement dans la barre supérieure.
Nouvelle option pour prendre en compte ou ignorer les titres sans contenu. Ils sont ignorés par défaut (dans les versions précédentes, ils étaient toujours pris en compte).
Les dialogues (pour les paramètres, l'aide, etc.) ont légèrement changé dans leur façon d'être affichés.
Cette version comprend également d'autres modifications pour améliorer la façon dont la fonctionnalité est injectée et exécutée. Des tests ont été effectués pour vérifier que tout fonctionnait comme prévu, mais il est possible que j'ai manqué des cas qui pourraient se produire. Si vous rencontrez un comportement ou des résultats inattendus, veuillez me contacter par e-mail à l'adresse headingsmap@gmail.com.
